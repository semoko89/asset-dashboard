업비트 실시간 시세

업비트 공개 API는 별도 키 없이 자동 연동됩니다. 코인 티커(예: BTC, ETH)를 자산 목록에 추가하면 자동으로 시세를 가져옵니다.

연동 API: https://api.upbit.com/v1/ticker
갱신 주기: 30초 자동 / 수동 새로고침 가능

국내주식 실시간 시세

국내주식은 네이버 금융 API를 통해 자동으로 시세를 가져옵니다. 별도 API 키가 필요 없습니다.

데이터 출처: 네이버 금융 (fchart.stock.naver.com)
티커 형식: 6자리 종목코드 (예: 005930 삼성전자, 000660 SK하이닉스)
갱신 주기: 30초 자동 / 수동 새로고침 가능
● 무료 · 키 불필요 — 자산 목록에 국내주식을 추가하면 자동으로 시세를 가져옵니다

해외주식 시세 (Twelve Data)

해외주식(미국 등)은 Twelve Data API로 시세를 가져옵니다. 무료 플랜으로 분당 8회 / 월 800회 제공됩니다.

1. twelvedata.com 에서 무료 가입 (이메일만)
2. 대시보드에서 API Key 복사
3. 아래 입력란에 붙여넣고 저장
● 무료 플랜: 분당 8회 / 월 800회 — 30초 간격 갱신 기준 하루 약 160회 사용 해외주식 티커: 미국 종목 심볼 입력 (예: AAPL, NVDA, TSLA)
● 키가 저장되어 있습니다.

데이터 저장

모든 자산 데이터는 이 기기의 브라우저(localStorage)에 저장됩니다. 브라우저를 바꾸거나 캐시를 지우면 초기화됩니다. 아래에서 데이터를 내보내거나 가져올 수 있어요.

GitHub Pages 배포 가이드

이 앱을 GitHub Pages에 올려서 언제 어디서나 접근하세요.

1. GitHub 계정에서 새 저장소 생성 (예: asset-dashboard)
2. 이 index.html 파일을 저장소에 업로드
3. Settings → Pages → Source: main branch / root 선택
4. https://[username].github.io/asset-dashboard 로 접근
5. 📌 브라우저 북마크 추가 or 모바일 홈화면에 추가
